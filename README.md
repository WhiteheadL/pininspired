# pininspired
A Pinterest-inspired photo sharing app - First full-stack portfolio project.
I'm building to learn full-stack web development from the ground up.

## Project status

🚧 **In active development** — started 17/04/2026. See [docs/requirements.md](docs/requirements.md) 
for planned features.

## What it does (planned for v1)

- Account creation with email, unique username, and securely stored password
- User profiles showing username, follower/following counts, and uploaded photos
- Photo uploads (PNG/JPEG, up to 5MB) with descriptions and tags
- Follow/unfollow other users
- Personal feed of photos from users you follow (newest first)
- Discover feed of most-liked photos when your feed is empty
- Like photos
- Search users by username and photos by tag
